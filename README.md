# About the project

This is a simple grocery list manager made with:

- Node.js
- React Native
- Sqlite

With this project you can:

- add, remove and update lists.
- add, remove, update, check and uncheck items.

## Prerequisites

You will need to have:

- Node.js 12.16.3
- NPM 6.14.4
- Expo 4.3.4

## Installation

Install node dependencies on backend  
`cd api & npm install`

Install node dependencies on application  
`cd app & npm install`

## Usage

Start the backend  
`npm run dev`

Start the application
`npm run start`
